# Trees Practice Portfolio

-This folder contains **tree-related Data Structures problems** implemented in Python.  
-It demonstrates understanding of **binary trees, binary search trees, and tree manipulations**, suitable for coding interviews and resume showcase.

---

## Objective
-Practice and showcase **tree algorithms** and operations such as traversal, inversion, and search.

---

## Problems Included
- **Invert Binary Tree** – Reverse the structure of a binary tree  
- **Binary Search Tree Operations** – Insert, Search, Delete nodes, and validate BST  

---

## Usage
1. Clone the repository or navigate to this folder.
```bash
git clone <repo-url>
cd trees
```
2. Run any problem script:
3. ```bash
   python <problem_name>.py
   ```

Notes:
-All scripts include clear comments for readability.
-Problems demonstrate tree manipulation, traversal, and binary search tree logic.
-This folder is professional and internship-ready, highlighting your data structure expertise.
