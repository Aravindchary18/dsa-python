This folder contains problems solved using the sliding window technique.

Sliding window is an efficient approach used to process subarrays or substrings without repeatedly recalculating results.

Concepts Practiced

- fixed window size
- dynamic window size
- substring analysis
- optimization of nested loops

Problems Implemented

1. "max_sum_subarray_k.py" – finds the maximum sum of a subarray of size k using sliding window.

2. "longest_substring_without_repeating.py" – finds the length of the longest substring without repeating characters using sliding window.
