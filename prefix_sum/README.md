This folder contains problems solved using the prefix sum technique.

Prefix sums are useful for efficiently calculating sums of subarrays and appear in many optimization problems.

Concepts Practiced

- cumulative sums
- hash map optimization
- subarray sum detection

Problems Implemented

1. "count_subarrays_sum_k.py" – counts the number of subarrays whose sum equals k using prefix sum and hash map.

2. "longest_subarray_sum_k.py" – finds the length of the longest subarray with sum equal to k.
