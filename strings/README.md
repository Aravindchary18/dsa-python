This folder contains common string manipulation problems frequently asked in coding interviews.

These problems focus on basic string operations such as character traversal, reversing, frequency counting, substring searching, and pattern checking.

Practicing these helps build a strong foundation for solving more advanced string and sliding window problems.

---

Concepts Covered

- String traversal
- Character frequency counting
- Palindrome checking
- String reversal
- Substring searching
- Basic string compression

---

Programs

1. "anagram.py" – checks whether two strings are anagrams of each other.

2. "count_frequency.py" – counts how many times each character appears in a string.

3. "palindrome.py" – checks if a string reads the same forward and backward.

4. "reverse_strings.py" – reverses the characters of a string.

5. "string_compression.py" – compresses consecutive repeating characters in a string.

6. "substring_search.py" – checks whether a substring exists inside another string.

---

Language

All solutions are implemented in Python.
