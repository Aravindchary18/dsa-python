-This folder contains common array-based problems frequently asked in coding interviews.

These problems help build skills in:

- Traversal techniques
- Two-pointer strategies
- Hash map optimization
- Sliding window
- Prefix sum

Practicing these problems improves problem-solving speed and prepares for coding interviews and assessments.

---

Concepts Covered

- Array traversal and manipulation
- Two-pointer techniques
- Prefix sum and cumulative sums
- Hash maps for counting/frequency
- Optimized subarray algorithms

---

Programs

1. "best_time_buy_sell_stock.py" – calculates the maximum profit from buying and selling stock once.

2. "container_with_most_water.py" – finds the maximum area of water that can be contained between two lines.

3. "majority_elements.py" – finds the element appearing more than n/2 times in the array.

4. "max_subarray.py" – finds the maximum sum of a contiguous subarray using Kadane’s Algorithm.

5. "merge_sorted_arrays.py" – merges two sorted arrays into one sorted array.

6. "move_zeroes.py" – moves all zero elements to the end while maintaining order.

7. "product_except_self.py" – returns an array where each element is the product of all other elements.

8. "remove_duplicates.py" – removes duplicate elements from a sorted array.

9. "three_sum.py" – finds all unique triplets in an array that sum to zero.

10. "two_sum.py" – finds two numbers in an array that add up to a target value using a hash map.

---

Language

All solutions are implemented in Python.
