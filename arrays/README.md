This folder contains solutions to common array-based problems.

Arrays are one of the most fundamental data structures and frequently appear in coding interviews. These problems help build intuition for:

- traversal techniques
- prefix computations
- two-pointer approaches
- optimization strategies

Problems Implemented

1. "two_sum.py" – finds two numbers in an array that add up to a target value using a hash map.

2. "max_subarray.py" – finds the maximum sum of a contiguous subarray using Kadane’s Algorithm.

3. "best_time_buy_sell_stock.py" – calculates the maximum profit from buying and selling stock once.

4. "move_zeroes.py" – moves all zero elements to the end of the array while maintaining order.

5. "merge_sorted_arrays.py" – merges two sorted arrays into one sorted array.

6. "remove_duplicates.py" – removes duplicate elements from a sorted array.

7. "product_except_self.py" – returns an array where each element is the product of all other elements.

8. "majority_elements.py" – finds the element that appears more than n/2 times in the array.

9. "container_with_most_water.py" – finds the maximum area of water that can be contained between two lines.

10. "three_sum.py" – finds all unique triplets in an array that sum to zero.
    
Each solution focuses on writing clean and efficient Python implementations.
