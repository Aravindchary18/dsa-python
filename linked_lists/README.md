Linked Lists

This folder includes implementations of common linked list problems.

Linked lists help understand pointer manipulation and are widely used in coding interviews.

Concepts Practiced

- list traversal
- reversing a linked list
- detecting cycles
- merging lists
- finding middle elements
- removing nodes

Problems Implemented

- Reverse Linked List
- Detect Cycle
- Middle of Linked List
- Merge Two Sorted Lists
- Remove Nth Node From End
- Palindrome Linked List
- Add Two Numbers
