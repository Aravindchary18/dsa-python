This folder contains common linked list problems frequently asked in coding interviews.

These problems help you practice:

- Pointer manipulation
- Node traversal
- Detecting cycles
- Reversing and merging lists
- Removing or adding nodes
- Finding middle elements

Mastering these builds a strong foundation for more advanced data structures.

---

Concepts Covered

- Singly linked list operations
- Reversing a linked list
- Detecting cycles using fast/slow pointers
- Merging sorted linked lists
- Removing nodes at specific positions
- Palindrome checking

---

Programs

1. "add_numbers.py" – adds two numbers represented by linked lists.

2. "detect_cycle.py" – detects whether a linked list contains a cycle.

3. "is_palindrome.py" – checks whether a linked list is a palindrome.

4. "merge_sorted_lists.py" – merges two sorted linked lists into one sorted list.

5. "middle_linked_list.py" – finds the middle node of a linked list.

6. "remove_Nth_from_end.py" – removes the Nth node from the end of the linked list.

7. "reversed_linked_lists.py" – reverses a singly linked list.

---

Language

All solutions are implemented in Python.
